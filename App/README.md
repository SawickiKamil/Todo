# TodoApp

### Front-End Tech

- [Vue.js]
- [TypeScript]
- [Vuex]
- [SCSS]

### Back-End Tech

- [JS]
- [Node.js]
- [Express]


### Front-End Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd Todo/FrontEnd
$ npm install
$ npm run start
```

### Back-End Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd Todo/BackEnd
$ npm install
$ npm run start || run nodemon
```
